
# React Todo CRUD Application

## Overview
A component-based Todo management application built with React.  
This project demonstrates state management using React Hooks and implements full CRUD functionality.

## Technologies Used
- React (Vite)
- JavaScript (ES6+)
- React Hooks (useState)
- React Icons
- CSS3

## Features
- Create new todos
- Update existing todos
- Delete todos
- Conditional rendering for edit mode
- Component-based architecture

## Technical Highlights
- Centralized state management in App component
- Props-based component communication
- Functional component structure
- Dynamic UI updates

## Installation

1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`
